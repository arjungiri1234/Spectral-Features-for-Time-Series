## How to Run
1. Clone the repository
2. Install dependencies:
bash
pip install numpy scipy matplotlib PyWavelets scikit-image
3. Open `Spectral-features.ipynb` in VS Code or Jupyter
4. Run all cells

---


## Topics Covered
- **FFT** — Finding hidden frequencies, frequency domain filtering,
  2D image spectrum analysis
- **STFT** — Spectrogram analysis, time-frequency tradeoff,
  Heisenberg uncertainty principle
- **Wavelet Transform** — VisuShrink denoising, CWT transient
  detection, DWT image compression
